# Page Structures by Product Type

## SaaS / Digital Tool (Full-Length)

Optimal for: Software products, tools, platforms with multiple features

```
1. Announcement Bar (optional)
   - Urgency/offer: "ðŸ”¥ 50% off first 3 months - Ends Friday"

2. Hero Section
   - H1: Specific outcome headline (36-60px desktop, 32-40px mobile)
   - H2: How you deliver the outcome
   - Visual: Product screenshot/demo GIF (not abstract illustration)
   - Primary CTA: High-contrast button
   - Social proof snippet: "Trusted by 10,000+ teams"

3. Logo Bar
   - 4-6 recognizable client logos
   - Label: "Trusted by teams at..."

4. Problem Agitation
   - Title: Present the broken status quo
   - 3 pain points with icons
   - Emotional language from customer research

5. Solution Introduction
   - Title: Position as the fix
   - 3 benefits mapped to pain points
   - Screenshot showing the solution

6. How It Works
   - 3 steps maximum
   - Icons + short descriptions
   - Optional: mini-demo video

7. Features (Bento Grid)
   - 4-6 feature cards
   - Each: Icon + Title + 1-sentence benefit
   - Highlight top feature with larger card

8. Social Proof Wall
   - 3-5 video testimonials (vertical, TikTok-style)
   - Or: Screenshot testimonials from Twitter/LinkedIn
   - Include: Name, photo, company, specific result

9. Comparison Table (Us vs Them)
   - Compare against 2-3 alternatives
   - Checkmarks for your features
   - X marks for competitors' gaps

10. Pricing Section
    - 3 tiers (Starter/Pro/Enterprise)
    - Highlight middle tier as "Most Popular"
    - Toggle: Monthly/Annual (show "2 months free")
    - CTA under each tier

11. FAQ Section
    - 6-8 common objections as questions
    - Collapsible accordion format
    - Include: pricing, refunds, support, integrations

12. Founder's Note (for Micro-SaaS)
    - Personal story
    - Photo of founder
    - Why you built this

13. Final CTA Block
    - Repeat main value prop
    - High-contrast CTA button
    - Trust badges (security, money-back guarantee)

14. Footer (Minimal)
    - Legal links only
    - Contact email
    - Social icons (optional)
```

## Course / Coaching (Story-Driven)

Optimal for: Online courses, coaching programs, memberships

```
1. Hero Section
   - H1: Transformation promise
   - H2: Who this is for + timeframe
   - Video: Creator intro (30-60 seconds)
   - CTA: "Join the Program" / "Enroll Now"

2. Pain Points (Story Format)
   - "If you're struggling with..."
   - 4-5 relatable pain points
   - Use customer's exact words

3. Your Story
   - Personal journey
   - Credentials/results
   - Photo of you teaching/working

4. The Transformation
   - Before â†’ After framework
   - Specific outcomes with numbers
   - "After this program, you'll..."

5. Curriculum / What's Inside
   - Module breakdown
   - Key lessons in each module
   - Time commitment per module

6. Bonuses Section
   - 3-5 extra resources
   - Assign value to each bonus
   - "Total value: $X,XXX"

7. Student Results
   - Video testimonials (best)
   - Screenshots of wins
   - Before/after metrics

8. Pricing
   - Payment plans if high-ticket
   - What's included list
   - Risk reversal (guarantee)

9. FAQ
   - "Is this for beginners?"
   - "How much time do I need?"
   - "What if it doesn't work?"

10. Urgency/Scarcity
    - Limited spots
    - Enrollment deadline
    - Price increase date

11. Final CTA
    - Reminder of transformation
    - Last call messaging
```

## Ebook / Lead Magnet (Short-Form)

Optimal for: Free downloads, checklists, templates, guides

```
1. Hero Section
   - H1: What they'll get + result
   - H2: Format (PDF, checklist, template)
   - Cover mockup image
   - Simple form: Email only (or Email + Name)

2. What's Inside
   - 3-5 key takeaways
   - Bullet points
   - Page count / time to consume

3. Who It's For
   - "Perfect for you if..."
   - 3 audience descriptions

4. Author Credibility
   - Brief bio
   - Key credentials
   - Small photo

5. Social Proof (Light)
   - "Downloaded by 5,000+ people"
   - Or: 1-2 short testimonials

6. Final CTA
   - Repeat the offer
   - Same simple form
```

## Newsletter (Ultra-Short)

Optimal for: Email newsletters, weekly digests, curated content

```
1. Hero Section
   - H1: What they'll learn/receive
   - H2: Frequency + value prop
   - Form: Email only
   - CTA: "Subscribe Free" / "Join 10,000 readers"

2. Sample Content
   - Screenshot of actual newsletter
   - Or: 3 recent issue titles

3. Social Proof
   - Subscriber count
   - Or: 2 short testimonials
   - Or: "Featured in [media logos]"

4. About the Creator
   - 2-3 sentence bio
   - Small photo
   - Optional: Social links
```

## Mobile Structure Adjustments

1. Stack all sections vertically
2. Sticky CTA at bottom (not header)
3. Collapsible FAQ (saves space)
4. Reduce testimonials to 3 max
5. Larger tap targets (48px minimum)
6. Remove horizontal logo scrollers
