## Hero Section (20 points)

### Headline (8 points)
- [ ] Headline promises a specific outcome or solves a pain point
- [ ] Headline is clear without industry jargon
- [ ] Headline matches the ad/search that brought the user
- [ ] Headline is 6-12 words (not too long)
- [ ] Headline addresses visitor's intent directly
- [ ] Headline uses emotional or benefit-driven language
- [ ] Subheadline expands on "how" (not just restating headline)
- [ ] Typography is impactful (36-60px desktop, proper weight)

### Visual (6 points)
- [ ] Product screenshot or demo is visible (not abstract illustration)
- [ ] Visual shows the product in use or the outcome
- [ ] Image/video loads quickly (<2s)
- [ ] Visual supports the headline message
- [ ] Mobile: Visual doesn't push CTA below fold
- [ ] No generic stock photos of smiling people

### Above-the-Fold CTA (6 points)
- [ ] Primary CTA is visible without scrolling
- [ ] CTA button has high contrast against background
- [ ] CTA text is action-oriented (not "Submit" or "Learn More")
- [ ] CTA text is personalized ("Get My..." vs "Get Your...")
- [ ] Only ONE primary CTA (not multiple competing actions)
- [ ] Micro-copy under CTA addresses objection (e.g., "No card required")

---

## Value Proposition (15 points)

### Clarity (8 points)
- [ ] What the product does is clear within 5 seconds
- [ ] Who it's for is immediately obvious
- [ ] Key benefit is stated, not just features
- [ ] Language matches how customers describe their problem
- [ ] No buzzwords ("revolutionary", "game-changing", "synergy")
- [ ] Specific numbers/metrics when possible
- [ ] A 10-year-old could understand the offer
- [ ] Value prop is unique (not same as competitors)

### Problem/Solution (7 points)
- [ ] Problem is explicitly stated before solution
- [ ] Pain points use customer's exact language (VoC)
- [ ] Consequences of not solving problem are clear
- [ ] Solution is positioned as direct answer to problem
- [ ] Benefits are mapped to each pain point
- [ ] "How it works" is explained simply (3 steps max)
- [ ] Features are translated into outcomes

---

## Social Proof (15 points)

### Credibility Signals (6 points)
- [ ] Client logos visible above fold
- [ ] Logos are recognizable/relevant to audience
- [ ] User count or usage stats displayed
- [ ] Awards/certifications shown (G2, Capterra, SOC2)
- [ ] Media mentions ("As featured in...")
- [ ] Logos have context ("Trusted by 5,000+ teams")

### Testimonials (9 points)
- [ ] At least 3 testimonials present
- [ ] Testimonials include full names and photos
- [ ] Company/role included for B2B
- [ ] Testimonials mention specific results/metrics
- [ ] At least one video testimonial present
- [ ] Testimonials are verifiable (links to LinkedIn/Twitter)
- [ ] Testimonials address common objections
- [ ] Variety of testimonial types (text, video, screenshot)
- [ ] No anonymous or vague testimonials ("Great product! - J.")

---

## CTA Optimization (15 points)

### Button Design (6 points)
- [ ] CTA color is used ONLY for CTAs (nowhere else)
- [ ] Button is large enough (48px+ height)
- [ ] CTA is repeated throughout page (not just hero)
- [ ] Sticky CTA appears on scroll (especially mobile)
- [ ] Button has hover/focus state
- [ ] Mobile: CTA is in thumb zone or sticky at bottom

### CTA Copy & Friction (9 points)
- [ ] CTA text describes the action outcome
- [ ] CTA text is personalized ("My" vs "Your")
- [ ] Form asks for minimal fields (<4)
- [ ] No unnecessary required fields
- [ ] Risk reversal present (guarantee, free trial)
- [ ] Trust badges near CTA (security, payment)
- [ ] Clear next step after CTA (what happens when I click?)
- [ ] No CAPTCHA or unnecessary friction
- [ ] Multi-step form for longer signups (reduces overwhelm)

---

## Copy Quality (15 points)

### Readability (7 points)
- [ ] Short paragraphs (2-3 sentences max)
- [ ] Bullet points for scannable content
- [ ] Headers break up long sections
- [ ] No walls of text
- [ ] Font size is readable (16px+ body)
- [ ] Sufficient line height (1.5-1.7)
- [ ] High contrast text (accessible)

### Persuasion (8 points)
- [ ] Benefits before features throughout
- [ ] Urgency or scarcity (if authentic)
- [ ] Objections addressed in FAQ
- [ ] Comparison to alternatives (if relevant)
- [ ] Founder's story (for smaller brands)
- [ ] Clear pricing (no "Contact us" only)
- [ ] Guarantee or risk reversal stated
- [ ] Final CTA with summary of offer

---

## Design & UX (10 points)

### Visual Design (5 points)
- [ ] Consistent color palette (3-4 colors max)
- [ ] Typography hierarchy is clear
- [ ] Sufficient whitespace (not cramped)
- [ ] Visual flow guides eye to CTA
- [ ] No clashing colors or outdated design

### Navigation & Structure (5 points)
- [ ] No main site navigation (or minimal)
- [ ] Only one exit point (the CTA)
- [ ] Logical section order (problem ‚Üí solution ‚Üí proof ‚Üí CTA)
- [ ] Anchor links for long pages (optional)
- [ ] Footer is minimal (doesn't distract)

---

## Technical Performance (10 points)

### Speed (5 points)
- [ ] Page loads in <3 seconds
- [ ] First contentful paint <1.5s
- [ ] Images are optimized (WebP/AVIF)
- [ ] No layout shift during load
- [ ] Lazy loading for below-fold content

### Mobile (5 points)
- [ ] Fully responsive (not just resized)
- [ ] Touch targets are 48px+
- [ ] No horizontal scroll
- [ ] Forms are easy to fill on mobile
- [ ] Page works on slow 3G connection

---

## Scoring Guide

| Score | Rating | Action |
|-------|--------|--------|
| 90-100 | Excellent | Minor optimizations only |
| 75-89 | Good | Address high-priority items |
| 60-74 | Average | Significant improvements needed |
| 40-59 | Poor | Major redesign recommended |
| <40 | Critical | Full rebuild required |
